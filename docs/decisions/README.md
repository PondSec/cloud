# Architecture Decision Records (ADRs)

This folder contains ADRs for significant technical decisions (new services, new dependencies, security posture changes, data model changes, etc).

## Naming

- `ADR-0001-<short-title>.md`
- Use the next available number.

## Required Sections

- Context
- Decision
- Alternatives Considered
- Security / Threat Impact
- Operational Impact (SRE)
- Migration / Rollback
- Test Plan
